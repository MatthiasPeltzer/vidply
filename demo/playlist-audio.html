Features Demonstrated

  • Previous/Next buttons - Navigate between tracks
  • Auto-advance - Automatically plays the next track
  • Track info display - Shows current track number, title, and artist
  • Playlist panel - Click any track to play it
  • Active track highlighting - Shows which track is currently playing
  • Keyboard navigation - Use Tab and Enter to navigate the playlist

How to Use

This demo shows how to create an audio playlist with VidPly:

  1. Create a player instance with new Player()
  2. Create a playlist manager with new PlaylistManager(player)
  3. Load your tracks with loadPlaylist()
  4. The previous/next buttons and track info will appear automatically
